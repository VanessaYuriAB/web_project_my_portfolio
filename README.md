# 📱 Meu Primeiro Portfólio — Projeto de Estudo em HTML e CSS

Este repositório contém o desenvolvimento de um projeto de estudo, meu primeiro portfólio
pessoal, **iniciado durante a Sprint 5 (Design Web Adaptativo) do bootcamp de**
**Desenvolvimento Web da TripleTen Brasil**.

Foi **continuado após a conclusão do curso e finalizado**, incorporando melhorias
estruturais, de organização de código e de boas práticas.

Documenta a **evolução do aprendizado inicial em front‑end e a aplicação progressiva de**
**boas práticas**, contendo os quatro projetos independentes iniciais do curso,
desenvolvidos ao longo das quatro primeiras Sprints.

Foi, inicialmente, estruturado com abordagem desktop-first e, posteriormente, refatorado
para abordagem **mobile-first**.

## 🎯 Objetivo do Projeto

O objetivo principal deste projeto foi praticar e consolidar fundamentos sólidos de
**desenvolvimento front‑end**, com foco em:

- **HTML semântico** e acessível
- **CSS moderno** e bem estruturado
- Aplicação da **metodologia BEM / BEM Flat**
- Organização de layout e estilos de forma **modular**
- **Responsividade e adaptação** a diferentes tamanhos de tela

---

## 🔄 Evolução do Projeto

### 📌 Fase inicial (durante o curso)

- Desenvolvimento **iniciado com abordagem desktop‑first**
- Apenas estruturação de conteúdo, com base em template do Figma
- Implementação da metodologia BEM (Flat)
- Recursos limitados ao escopo do curso (HTML e CSS)

### 🚀 Continuação após a conclusão do curso

Após a finalização do bootcamp, o projeto foi retomado e evoluiu significativamente:

- Melhoria da hierarquia semântica do HTML
- Refatoração completa da base de estilos
- Conversão gradual para uma **abordagem mobile‑first**
- Organização mais clara do CSS por responsabilidade
- Centralização de estilos reutilizáveis (ex.: botões e títulos de seções)
- Foco em layout adaptativo, media queries e organização visual
- Ajustes finos de tipografia, espaçamento e layout
- **Padronização de unidades** (`rem`, `em`, `px`) conforme boas práticas
- Evolução progressiva do código a partir de refatorações e feedback (Copilot)

Além disso, foram introduzidas **animações em CSS** puro com foco em leveza e performance,
incluindo:

- Implementação de animações de revelação (_reveal_) de conteúdo com `clip-path`
- Controle de fluxo visual por meio de animações direcionais (ex.: esquerda → direita e
  direita → esquerda)

Essa fase refletiu o amadurecimento técnico adquirido ao longo do curso e a aplicação
consciente de conceitos como reuso, legibilidade e manutenção de código.
