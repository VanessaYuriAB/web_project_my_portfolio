# Primeiro Portfólio – Projeto de Estudo

Este repositório contém a primeira versão do meu portfólio pessoal, iniciada durante a
**Sprint 5 – Design Web Adaptativo** do bootcamp de **Desenvolvimento Web da TripleTen**
**Brasil**.

A proposta deste projeto foi **praticar e consolidar conhecimentos de HTML e CSS**, a
partir de templates de design e boas práticas de organização de código, sem a exigência
de finalização completa ou publicação profissional.

---

## 🎯 Objetivo do Projeto

- Aplicar **HTML semântico**
- Estruturar layouts com **CSS avançado**
- Utilizar a metodologia **BEM e BEM Flat**
- Trabalhar com **design responsivo**
- Organizar o projeto seguindo um fluxo de desenvolvimento modular

Este projeto foi iniciado como **atividade complementar sugerida** nos recursos adicionais
da Sprint 5.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Metodologia BEM / BEM Flat
- Organização modular de estilos
- Template de design do Figma

---

## 📚 Contexto no Curso

Na Sprint 5, o foco esteve em:

- Design web adaptativo
- Layouts responsivos
- Media queries
- Boas práticas de estruturação visual

O curso seguiu evoluindo para novos conteúdos nos sprints seguintes, e este projeto foi
mantido como **registro do aprendizado inicial**, não como produto final.

---

## 🚧 Status do Projeto

🔸 **Projeto em desenvolvimento / não finalizado**

Algumas seções, textos e funcionalidades permanecem como rascunho ou placeholder.

Atualmente, este repositório é mantido como **projeto de estudo**, enquanto versão mais
recente de portfólio será desenvolvida separadamente.

---

## 🌱 Observação

Este projeto representa uma **etapa inicial da minha jornada como desenvolvedora web** e
serve como referência de aprendizado, organização de código e evolução técnica ao longo do
bootcamp.
